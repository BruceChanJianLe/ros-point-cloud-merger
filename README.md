# HOW TO USE THIS PACKAGE

## Launch package

```
roslaunch ros-point-cloud-merger point_cloud_merger.launch
```
>By default, configuration would be husky.yaml

>Configuration can be accessed in ros-point-cloud-merger/config/husky.yaml
